Wandel Widget zur API um

*Daten von Fußball.de

Wandle dein Tabellen-Widget von Fußball.de in eine API um. Als Ergebnis erhältst du einen Link, der mittels HTTP-Request eine JSON-Liste der Tabelle zurückgibt. Diese enthält den Tabellenplatz, den Vereinsnamen, das Logo als Link, die Punkte, die Tordifferenz und die Anzahl der gespielten Spiele.
